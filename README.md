# Dady Proxy — Render.com Deploy

## Deploy Adımları

1. Bu klasörü GitHub'a yükle (yeni repo oluştur)
2. [render.com](https://render.com) → New → Web Service
3. GitHub reposunu bağla
4. Ayarlar otomatik `render.yaml`'dan gelir → **Deploy** bas

## Kullanım

```
https://RENDER_URL/?ID=51
https://RENDER_URL/?CDN=<m3u8_url>
```

## M3U Güncelleme

`Dady_worker.m3u` içindeki URL'leri Render URL'inle değiştir.
